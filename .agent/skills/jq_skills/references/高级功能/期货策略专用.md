# 期货策略专用指南

## 1. 期货交易核心 API

### 下单指令
- `order(security, amount)`: 股数/手。正数为多头，负数为空头。
- **注意**：期货是保证金交易，`context.portfolio.available_cash` 会随着保证金占用而减少。

### 保证金设置
- 默认按交易所标准。可以使用 `set_margin_rate` 调整。

---

## 2. 定时任务与参考标的 (重要)
期货有夜盘，因此 `run_daily` 的 `reference_security` 必须正确。

**错误示例**：
`run_daily(my_func, time='21:00')` -> 如果参考默认的 '000001.XSHG'，由于股票 21:00 不开盘，此函数将**不会运行**。

**正确示例**：
```python
# 参考螺纹钢，在夜盘开盘时运行
run_daily(my_func, time='open', reference_security='RB9999.XSGE')
```

---

## 3. 期货持仓管理
期货持仓区分多单和空单，通过 `context.portfolio.positions` 访问时：
- 多单：`amount > 0`
- 空单：`amount < 0` (在旧版 API 中可能通过 `side` 区分，请务必检查 `long_positions` 和 `short_positions`)。

---

## 4. 结算价与交割
- **交割日处理**：期货持仓到交割日，系统会以当天**结算价**平仓，无手续费。
- **结算价(Settlement Price)**：回测中用于计算当日盈亏的价格，不同于收盘价。

---

## 5. 还券与融资融券 (T+1)
- 当日融的券，当日不能还。
- 直接还券可以使用当日买入的券。
