# 投资组合优化器 (全量指南)

## 1. 核心流程
```python
from jqlib.optimizer import portfolio_optimizer
res = portfolio_optimizer(date, securities, target, constraints, bounds)
```

## 2. 优化目标函数 (Targets)

| 函数 | 说明 |
| :--- | :--- |
| **RiskParity** | 风险平价。`RiskParity(count=250)` |
| **MinVariance** | 最小化组合方差。 |
| **MaxProfit** | 组合预期收益最大化。 |
| **MaxSharpeRatio** | 组合夏普比率最大化。`MaxSharpeRatio(rf=0.04)` |
| **MaxFactorValue** | 最大化特定因子。`MaxFactorValue(factor, count)` |
| **MinTrackingError** | 最小化相对于基准的追踪误差。 |

---

## 3. 约束条件 (Constraints)

| 类名 | 描述 | 示例 |
| :--- | :--- | :--- |
| **WeightConstraint** | 限制组合总权重范围。 | `WeightConstraint(0.9, 1.0)` |
| **MarketConstraint** | 按市场(stock/etf)限制。 | `MarketConstraint('etf', 0, 0.2)` |
| **IndustryDeviation** | 限制相对于基准的行业偏差。 | `IndustryDeviationConstraint(index, 0.05)` |
| **AnnualProfit** | 设定年化收益率下限。 | `AnnualProfitConstraint(0.1)` |
| **TurnoverConstraint** | 限制换手率。 | `TurnoverConstraint(0.2)` |
| **Bound** | 单只标的的权重上下限。 | `Bound(0, 0.1)` |

---

## 4. 常见问题处理
- **优化失败**：如果约束过于严格（如要求高收益但低波动），`portfolio_optimizer` 可能返回 `None`。
- **参数 `count`**：通常设为 250（一年），用于计算协方差矩阵。
