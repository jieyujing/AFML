# 历史行情数据深度指南

## 1. 常用行情获取函数

### attribute_history(security, count, unit='1d', fields=['open', 'close', 'high', 'low', 'volume', 'money'], skip_paused=True)
**描述**：获取单个标的最近 `count` 个单位的历史数据。
- **unit**: '1d' (天), '1m' (分钟), '5m', '15m', '30m', '60m', '120m'。
- **fields**: 支持 `['open', 'close', 'high', 'low', 'volume', 'money', 'factor', 'high_limit', 'low_limit', 'avg', 'pre_close']`。
- **skip_paused**: 是否跳过停牌日。默认 True。

---

### get_price(security, start_date=None, end_date=None, frequency='daily', fields=None, skip_paused=False, fq='pre', count=None)
**描述**：功能最全的行情函数。
- **fq**: 复权方式。'pre' (前复权), 'post' (后复权), `None` (不复权)。
- **注意**：在 `initialize` 设置了 `use_real_price` 之后，`fq` 默认自动处理。
- **count**: 与 `start_date` 二选一，获取 `end_date` 之前的 `count` 条数据。

---

### get_bars(security, count, unit='1d', fields=['date', 'open', 'close'], include_now=False, end_dt=None)
**描述**：比 `get_price` 更快的接口，返回 `numpy.ndarray`，适合大规模数据处理。

---

### get_fundamentals(query_object, date=None, statDate=None)
**描述**：获取基本面（财务、市值）数据。

**核心查询表**：
1. **valuation**: 市值、PE、PB。
2. **balance**: 资产负债表。
3. **cash_flow**: 现金流量表。
4. **income**: 利润表。
5. **indicator**: 财务指标（ROE, EPS等）。

**典型代码**：
```python
# 查询市值小于 50 亿且 ROE > 15% 的股票
from jqdata import valuation, indicator
q = query(valuation.code).filter(
    valuation.market_cap < 50,
    indicator.roe > 15
).order_by(valuation.market_cap.asc())
df = get_fundamentals(q)
```

## ⚠️ 避坑指南
- **停牌处理**：`paused` 属性为 `True` 时，当日成交量为 0。
- **数据对齐**：分钟 K 线中，`09:31` 的 K 线包含了 `09:25` 的集合竞价数据。
- **财务数据延迟**：财报数据在发布日之后才会在 `get_fundamentals` 中体现，默认已处理“前瞻性偏误”。
