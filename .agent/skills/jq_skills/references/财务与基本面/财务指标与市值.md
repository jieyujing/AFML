# 财务指标与市值

## 1. 基础财务查询

### get_fundamentals(query_object, date, statDate)
**描述**：获取基本面数据。

**参数**：
- `query_object`: 使用 `query()` 函数创建的查询对象，通常涉及 `valuation`, `balance`, `cash_flow`, `income`, `indicator` 五张表。
- `date`: 查询日期。
- `statDate`: 查询财报统计季度/年份。

**示例**：
```python
# 查询市盈率小于 10 的股票
q = query(valuation.code).filter(valuation.pe_ratio < 10)
df = get_fundamentals(q, date='2023-01-01')
```

## 2. 核心表结构

### valuation (市值表)
- `pe_ratio`: 动态市盈率
- `market_cap`: 总市值(亿元)
- `circulating_market_cap`: 流通市值(亿元)
- `pb_ratio`: 市净率

### indicator (财务指标表)
- `eps`: 每股收益
- `roe`: 净资产收益率
- `net_profit_margin`: 销售净利率

## 3. 资金流向

### get_money_flow(security_list, start_date, end_date)
**描述**：获取主力、大单、中单、小单的净额和净占比。
- `net_amount_main`: 主力净额(万)
