# 技术指标与分析库 (jqlib/TA-Lib)

聚宽提供了两套技术指标系统：针对 Pandas 优化的 `jqlib` 和高性能的 `TA-Lib`。

## 1. JQLib 技术指标
**导入**：`from jqlib.technical_analysis import *`

| 指标 | 接口 | 示例 |
| :--- | :--- | :--- |
| **MACD** | `MACD(security_list, date, SHORT, LONG, MID)` | 返回 `dif, dea, macd` 的 Series |
| **RSI** | `RSI(security_list, date, N1)` | 返回 RSI 值 |
| **KDJ** | `KDJ(security_list, date, N, M1, M2)` | 返回 `K, D, J` |
| **Bollinger** | `Bollinger_Bands(security_list, date, timeperiod, nbdevup, nbdevdn)` | 返回 `upper, middle, lower` |

---

## 2. TA-Lib 高性能库
**描述**：直接操作 Numpy 数组，速度极快。
```python
import talib
close = attribute_history(security, 100, '1d', ['close'])['close'].values
# 计算 SMA
sma20 = talib.SMA(close, timeperiod=20)
```

---

## 3. Alpha 因子计算
聚宽预装了 `alpha101` 和 `alpha191`。

- **Alpha101**: `from jqlib.alpha101 import *` -> `alpha_001(security_list, date)`
- **Alpha191**: `from jqlib.alpha191 import *` -> `alpha_001(security_list, date)`

---

## 4. 统计与数学函数 (常用)
- `stddev(data, timeperiod)`: 标准差。
- `covariance(data1, data2, timeperiod)`: 协方差。
- `correlation(data1, data2, timeperiod)`: 相关系数。
- `Rank(data)`: 截面排名。
