# 技术指标与因子库

## 1. TA-Lib 指标库
聚宽环境集成了标准的 TA-Lib，所有函数接受 `numpy.ndarray`。

### 示例：计算 RSI
```python
import talib
close_prices = attribute_history('000001.XSHE', 30, '1d', ['close'])['close'].values
rsi = talib.RSI(close_prices, timeperiod=14)
```

## 2. JQLib 技术指标库
针对中国市场优化的指标库。

| 函数 | 说明 |
| :--- | :--- |
| **MACD** | `MACD(security_list, check_date, SHORT=12, LONG=26, MID=9)` |
| **RSI** | `RSI(security_list, check_date, N1=6)` |
| **KDJ** | `KDJ(security_list, check_date, N=9, M1=3, M2=3)` |

---

## 3. Alpha101 / Alpha191 因子
聚宽提供了 WorldQuant 和国泰君安的因子库。

**导入方式**：
- `from jqlib.alpha101 import *`
- `from jqlib.alpha191 import *`

**调用**：`alpha_001(security_list, date)`

---

## 4. 聚宽因子库 (jqfactor)
提供高质量的风格因子和预计算指标。

**核心函数**：`get_factor_values(securities, factors, start_date, end_date)`

**常用因子名**：
- `size`: 市值因子
- `volatility`: 波动率因子
- `quality`: 质量因子
